# cars-sale-of-year-with-graphs
This is an Excel-based automotive sales dashboard analyzing car model performance by revenue and price. It compares 6 models—Honda City, Hyundai Creta, Kia Seltos, Mahindra Scorpio, Maruti Swift, and Tata Nexon—across 9 visualizations including bar charts, pie charts, line graphs, and a pivot table.
